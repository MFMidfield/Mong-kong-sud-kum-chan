โปรเจกต์นี้เป็นเว็บเพจเชิงการศึกษา นำเสนอ มงคลสูตร จากพระพุทธศาสนา ในรูปแบบ คำฉันท์และการอธิบายอย่างเป็นระบบ เพื่อช่วยให้เข้าใจเนื้อหาเชิงภาษา วรรณศิลป์ และคุณค่าทางชีวิตได้ง่ายขึ้น

วัตถุประสงค์ของโปรเจกต์
* เพื่อเผยแพร่คำสอนจาก มงคลสูตร ในรูปแบบที่อ่านเข้าใจง่าย
* เพื่อใช้เป็นสื่อประกอบการเรียนภาษาไทย วรรณคดี และพระพุทธศาสนา
* เพื่อผสานเทคโนโลยีเว็บกับเนื้อหาทางวัฒนธรรมและศาสนาอย่างเหมาะสม

เนื้อหาภายในเว็บไซต์
แต่ละคาถาประกอบด้วย:
1. คาถาบาลี
2. คำอ่านบาลี
3. คำศัพท์สำคัญ
4. การถอดความเป็นภาษาร่วมสมัย
5. การวิเคราะห์คุณค่า (วรรณศิลป์ / สังคม / เนื้อหา)
6. ข้อคิดและความรู้เพิ่มเติม

 เทคโนโลยีที่ใช้
* HTML5 – โครงสร้างเนื้อหาเว็บไซต์
* Tailwind CSS – การจัดรูปแบบและออกแบบ UI
* JavaScript (Vanilla JS) – เอฟเฟกต์และการโต้ตอบ
* AOS (Animate On Scroll) – แอนิเมชันขณะเลื่อนหน้า
* Google Fonts – ฟอนต์ภาษาไทย (Sarabun, Noto Serif Thai)

โครงสร้างไฟล์
/project-root
│── index.html   # หน้าเว็บหลัก
│── script.js    # สคริปต์ควบคุม animation และ interaction

วิธีการใช้งาน
1. ดาวน์โหลดหรือ clone โปรเจกต์นี้
2. เปิดไฟล์ index.html ผ่านเว็บเบราว์เซอร์
3. เลื่อนหน้าเพื่ออ่านเนื้อหาและชมแอนิเมชัน

This project is an educational web page presenting the Mangala Sutta from Buddhism, structured through Thai verse interpretation and systematic explanations to support understanding of language, literary aesthetics, and life values.

Project Objectives
* To present the Mangala Sutta in a structured and accessible format
* To serve as an educational resource for literature, language, and Buddhist studies
* To combine modern web design with cultural and religious content respectfully

Content Structure
Each verse includes:
1. Pali verse
2. Pali pronunciation
3. Key vocabulary explanations
4. Modern Thai interpretation
5. Value analysis (literary, social, and thematic)
6. Reflections and related knowledge

Technologies Used
* HTML5 – Website content structure
* Tailwind CSS – UI styling and layout
* JavaScript (Vanilla JS) – Interactions and effects
* AOS (Animate On Scroll) – Scroll-based animations
* Google Fonts – Thai fonts (Sarabun, Noto Serif Thai)

File Structure
/project-root
│── index.html   # Main webpage
│── script.js    # Animation and interaction logic

How to Use
1. Download or clone this repository
2. Open index.html in a web browser
3. Scroll through the page to explore the content and animations
